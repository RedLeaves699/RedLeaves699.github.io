    K - Stack

    算法 > 题解 2021牛客暑期多校第二场
    Created At : 2021-07-26 09:37
    Views 👀 :

    2. 题目大意
    3. 解题思路
    4. 编程实现

    题目原文

    YT had a magic permutation \(a_{1},a_{2},\cdots, a_{n}\), and he constructed a sequence \(b_{1},b_{2},\cdots b_{n}\) by the following pseudo code:

    Stk is an empty stack
for i = 1 to n :
    while ( Stk is not empty ) and ( Stk's top > a[i] ) :
        pop Stk
    push a[i]
    b[i]=Stk's size

    But he somehow forgot the permutation \(a\), and only got some \(k\) elements of \(b_{i}\).

    Construct a permutation \(a\) satisfying these \(b_{i}\), or determine no such permutation exists.

    Here a permutation refers to a sequence that contains all integers from \(1\) to \({n}\) exactly once.

    题目大意

    定义条件A为：位置\(pos\)之前（包括该位置）有\(x\)个数不小于这个位置上的数\(a[pos]\)。现在知道多个位置满足条件A,将这些要求记为\((pos, x)\)，能否构造一个序列，满足给定的全部要求？

    解题思路

    先考虑什么时候不能，不能的情况其实只有一种，就是\(b[i]\)的需求大于提供的\(i\)个位置。

    再考虑可行的情况下，如何解决安排的问题。由离散数学的知识，小于等于(或大于)是一个偏序，很多个小于等于情况可以抽象为偏序关系，继而可以得到关系图，偏序的关系图是有向无环的(DAG)，所以想到根据题目条件建立关系图后，用拓扑排序的思想解题。

    关系图的节点是谁？只能是位置，因为我们知道的条件是关于位置的。如何知道这个位置(上的数)和其他位置(上的数)的关系？可以模拟位置序列的插入。

    1234567...，将这些属于位置的编号，按照伪代码的方式压栈弹栈，注意，当某个位置有需求时，需要将栈的大小弹至b[i]规定的大小。此时，这个位置上的数必大于等于于栈顶元素，小于最后一次弹出的数，这样就建立了关系图， \[a[栈顶数的位置] \leq a[位置pos] < a[刚弹出的数的位置]\]

    可以抽象出这样的关系图(代码中的抽象关系)

    \[刚弹出的数的位置 \rightarrow pos \rightarrow 栈顶数的位置\]

    由此，拓扑排序第一层就是最大层，此后的数越来越小

    实现的详情见代码注释

    编程实现

    #include <cstdio>
#include <algorithm>
#include <queue>
#include <stack>
#include <vector>

const int MAXN = 1e6 + 10;
int b[MAXN], in[MAXN], next[MAXN], ans[MAXN];

void Print(std::vector <int> &v, int n) {
    for (int i = 0; i < v.size(); ++i) {
        ans[v[i]] = n - i;//在相应位置由大到小填入数字
    }
    for (int i = 1; i <= n ; ++i) {
        if (i == 1)printf("%d", ans[i]);
        else printf(" %d", ans[i]);
    }printf("\n");
}

std::queue <int> q;
std::vector <int> v;
void toposort(int n) {  
    for (int i = 1; i <= n; ++i) {
        if (in[i] == 0) q.push(i);
    }
    while (!q.empty()) {
        int t = q.front();q.pop();
        v.push_back(t);//按照拓扑序放置位置序
        in[next[t]]--;
        if (in[next[t]] == 0) {
            q.push(next[t]);
        }
    }
    Print(v, n);
}

std::stack <int> s;
int solve() {
    int n, k;
    scanf("%d%d", &n, &k);
    for (int i = 1; i <= k; ++i) {
        int pos;
        scanf("%d", &pos);scanf("%d", &b[pos]);
    }
    for (int i = 1; i <= n; ++i) {
        int t;
        if (b[i]) {
            if (b[i] > i) return -1;
            while (s.size() && s.size() >= b[i]) t = s.top(), s.pop();
            next[t] = i;
        }
        if(!s.empty()) next[i] = s.top();
        s.push(i);
    }
    for (int i = 1; i <= n; ++i) {
        in[next[i]]++;
    }
    toposort(n);
    return 0;
}
int main() {
    if (solve() == -1) puts("-1");
    return 0;
}
