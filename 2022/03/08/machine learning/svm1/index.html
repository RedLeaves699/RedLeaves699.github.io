    SVM的相关知识

    SVM中涉及的主要思想（三宝）：间隔，对偶，核技巧。

    SVM的主要类别：硬间隔，软间隔，核技巧。

    线性可分支持向量机

    解决的问题及思想

    给定训练集\(D=\{(\pmb{x}_i,y_i)\}_{i=1}^m,\pmb{x}_i=(x_1,x_2,\dots,x_d),y_i\in\{0,1\}\)，给定的训练集是线性可分的，如下图所示。寻找一个超平面尽可能正确分类测试集中的示例。图中红色表示正例，蓝色表示反例。

    利用误分类最小策略划分出的超平面有无数个，如上图展示了几个超平面。但最好的是哪一个超平面？直观上讲，两类训练样本最中间的平面（图上橙色的）该最好，即使用间隔最大化求出的超平面。这个超平面受噪声的影响最小，对未见示例得泛化能力应该最强。在样本空间中，划分超平面可通过如下线性方程来描述： \[ \pmb{w}^\text{T}\pmb{x}+b=0 \] 其中\(\pmb{w}=(w_1,w_2,\dots,w_n)^\text{T}\)为平面的法向量，决定了超平面的方向，并且指向正例，\(b\)为位移项，决定了超平面与原点之间的距离。

    定义

    给定线性可分数据集，通过间隔最大化等价的求解相应的凸二次规划问题学习得到的分离超平面为： \[ \pmb{w}^{*\text{T}}\cdot\pmb{x}+b^*=0 \] 以及相对应的分类决策函数： \[ f(\pmb{x})=\text{sign}(\pmb{w}^{*\text{T}}\cdot\pmb{x}+b^*) \]

    间隔最大化模型

    点到超平面的距离公式

    平面外一点到平面的距离公式为： \[ \gamma = \frac{|\pmb{w}^\text{T}\pmb{x}+b|}{||\pmb{w}||}\tag{6.2} \] 证明：设平面外一点为A，在平面上任取一点为C，\(\vec{AC}=\pmb{C}-\pmb{A}\)，\(\stackrel{\rightarrow}{AC}\)在\(\pmb{w}\)上的投影长度为\(\vec{AC}\cdot\pmb{w}=|\vec{AC}|\cdot||\pmb{w}||\cdot\cos<\vec{AC},\pmb{w}>\)，距离\(\gamma=|\vec{AC}|\cdot\cos<\vec{AC},\pmb{w}>\)，所以可推出： \[ \gamma = \frac{\vec{AC}\cdot\pmb{w}}{||\pmb{w}||} \] 得到距离公式(6.2)。

    几何间隔

    关于给定数据集\(D\)和超平面\(\pmb{w}^\text{T}\pmb{x}+b=0\)，定义数据集任意一个样本点\((\pmb{x}_i,y_i),y_i\in\{-1,1\}\)，关于超平面的几何间隔为： \[ \gamma_i=y_i\frac{(\pmb{w}^\text{T}\pmb{x}+b)}{||\pmb{w}||} \] 正确分类时，\(\gamma_i>0\)，错误分类时，\(\gamma_i<0\)。

    定义数据集\(D\)关于超平面的几何间隔为：数据集中所有样本点的几何间隔最小值。 \[ \gamma = \min \gamma_i \] 支持向量机转为约束条件下的优化问题： \[ \underset{\pmb{w},b}{\max} \gamma\\ s.t. \gamma_i \geq \gamma \] 设\(\pmb{x}_{min},y_{min}\)表示\(\gamma\)对应的数据，代入几何间隔计算公式，原问题可表示为： \[ \underset{\pmb{w},b}{\max} \gamma = y_{min}\frac{\pmb{w}^\text{T}\pmb{x}_{min}+b}{||\pmb{w}||}\\ s.t.\ y_i(\pmb{w}^\text{T}\pmb{x}_i+b) \geq y_{min}(\pmb{w}^\text{T}\pmb{x}_{min}+b) \]

    假设该问题的最优解是\((\pmb{w}^*,b^*)\)，那么\((\alpha\pmb{w}^*,\alpha b^*),\alpha\in\mathbb{R}^+\)也是最优解，且超平面不变。因此需要人为的对\((\pmb{w}^*,b^*)\)，做一定的限制，才能是这个最优化问题有可解的唯一解。不妨令\(\gamma=1\)，这样对于特定的\((\pmb{x}_{min},y_{min})\)来说，令\(\gamma=1\)的\(\alpha\)有且仅有一个。

    我们的问题化解为： \[ \underset{\pmb{w},b}{\max} \gamma = \frac{1}{||\pmb{w}||}\\ s.t.\ \ y_i(\pmb{w}^\text{T}\pmb{x}_i+b) \geq 1\tag{6.3} \] 我们现在构建的模型如下图所示：

    求\(\gamma\)的最大，就相当于求\(\frac{1}{\gamma}\)的最小，将(6.3)改为最优化问题的常见形式： \[ \underset{\pmb{w},b}{\min} \frac{1}{2}||\pmb{w}||^2\\ s.t.\ 1 - y_i(\pmb{w}^\text{T}\pmb{x}_i+b) \leq 0 \] 这就是支持向量机的基本型。上面的间隔最大化模型，又称为硬间隔。

    求解模型

    直接求解可能的问题

    如果样本的维度较低，可以直接使用现有的软件包直接处理。但是，如果维度比较高，或者使用了核函数将示例从低维转化为高维，那么，暴力求解的开销将会非常大，效率就会大大降低。上述的优化问题有一个很好的性质，就是对偶性。使用对偶性，可以加速问题的求解。

    转化为无约束问题

    使用拉格朗日乘子法构造拉格朗日函数来去掉约束： \[ L(\pmb{w},b,\pmb{\alpha})=\frac{1}{2}||\pmb{w}||^2+\stackrel{m}{\underset{i=1}\sum}\alpha_i(1-y_i(\pmb{w}^\text{T}\pmb{x}_i+b))\tag{6.8} \] 其中\(\alpha_i>0\)(目标和约束函数梯度一定相反)。原问题相当于： \[ \underset{\pmb{w},b}{\min} \underset{\pmb{\alpha}}{\max} L(\pmb{w},b,\pmb{\alpha})\\ s.t.\ \pmb{\alpha} \succeq 0 \] 证明：

    1. 如果\(1-y_i(\pmb{w}^\text{T}\pmb{x}_i+b)>0\)，则\(\underset{\pmb{\alpha}}{\max}L(\pmb{w},b,\pmb{\alpha})=\frac{1}{2}||\pmb{w}^2||+\infty\)。

    2. 如果\(1-y_i(\pmb{w}^\text{T}\pmb{x}_i+b)\leq0\)，则\(\underset{\pmb{\alpha}}{\max}L(\pmb{w},b,\pmb{\alpha})=\frac{1}{2}||\pmb{w}^2||+0\)，得到\(\underset{\pmb{w},b}{\min} \underset{\pmb{\alpha}}{\max} L(\pmb{w},b,\pmb{\alpha})=\underset{\pmb{w},b}{\min} \frac{1}{2}||\pmb{w}^2||\)。

      综合以上两点： \[ \underset{\pmb{w},b}{\min} \underset{\pmb{\alpha}}{\max} L(\pmb{w},b,\pmb{\alpha})=\underset{\pmb{w},b}{\min}(\infty,\frac{1}{2}||\pmb{w}||^2)=\underset{\pmb{w},b}{\min}\frac{1}{2}||\pmb{w}||^2 \]

    转化为对偶问题

    原问题的\(\pmb{\alpha}\)是非常难求的。所以我们想通过先最优化\(\pmb{w},b\)，最后确定\(\pmb{\alpha}\)的方法解决现在的问题。于是我们引入了原问题的对偶问题。

    给出原问题的对偶问题： \[ \underset{\pmb{\alpha}}{\max} \underset{\pmb{w},b}{\min}L(\pmb{w},b,\pmb{\alpha})\\ s.t.\ \pmb{\alpha} \succeq 0 \] 对偶问题和原问题之间一定存在弱对偶关系： \[ \min \max L \geq \max \min L \] 最大值中最小的一定大于等于最小值中最大的。

    为了解决上面的问题，我们需要强对偶关系，上面的问题是凸优化问题，可以证明，凸优化问题一定满足强对偶关系： \[ \min \max L = \max \min L \] 下面就可以通过求解对偶问题得到原问题的解。令\(L(\pmb{w},b,\pmb{\alpha})\)分别对\(\pmb{w},b\)求导： \[ \begin{align*}\frac{\partial L(\pmb{w},b,\pmb{\alpha})}{\partial \pmb{w}}&=\frac{\partial \frac{1}{2}\pmb{w}^\text{T}\pmb{w}+\stackrel{m}{\underset{i=1}\sum}\alpha_i-\stackrel{m}{\underset{i=1}\sum}\alpha_iy_i\pmb{w}^\text{T}\pmb{x}_i+\stackrel{m}{\underset{i=1}\sum}\alpha_iy_ib}{\partial \pmb{w}}\\&=\pmb{w}-\stackrel{m}{\underset{i=1}\sum}\alpha_iy_i\pmb{x}_i\end{align*} \]

    \[ \begin{align*}\frac{\partial L(\pmb{w},b,\pmb{\alpha})}{\partial b}&=\frac{\partial \frac{1}{2}\pmb{w}^\text{T}\pmb{w}+\stackrel{m}{\underset{i=1}\sum}\alpha_i-\stackrel{m}{\underset{i=1}\sum}\alpha_iy_i\pmb{w}^\text{T}\pmb{x}_i+\stackrel{m}{\underset{i=1}\sum}\alpha_iy_ib}{\partial b}\\&=\stackrel{m}{\underset{i=1}\sum}\alpha_iy_i\end{align*} \]

    令导数等于0求得\(L(\pmb{w},b,\pmb{\alpha})\)的极值点： \[ \pmb{w}^*=\stackrel{m}{\underset{i=1}\sum}\alpha_iy_i\pmb{x}_i\tag{6.9} \]

    \[ \stackrel{m}{\underset{i=1}\sum}\alpha_iy_i=0\tag{6.10} \]

    将(6.9)(6.10)代入(6.8)，对偶问题可化为： \[ \underset{\pmb{\alpha}}{\max}\ -\frac{1}{2}\stackrel{m}{\underset{i=1}\sum}\stackrel{m}{\underset{i=1}\sum}\alpha_i\alpha_jy_iy_j\pmb{x}_i^\text{T}\pmb{x}_j+\stackrel{m}{\underset{i=1}\sum}\alpha_i\\s.t.\ \stackrel{m}{\underset{i=1}\sum}\alpha_iy_i=0\tag{6.11}\\ \pmb{\alpha}\succeq 0\\ \] \(\pmb{w}\)已经求解完成，那么使用SMO1算法解出\(\pmb{\alpha}\)后，再求出\(b\)即可得到模型： \[ f(\pmb{x})=\pmb{w}^\text{T}\pmb{x}+b=\stackrel{m}{\underset{i=1}\sum}\alpha_iy_i\pmb{x}_i^\text{T}\pmb{x}+b\tag{6.12} \]

    \(b\)该怎么求？

    使用KKT条件求解b的值

    可以证明一个结论，原问题和对偶问题具有强对偶关系的充要条件是它们满足KKT条件。

    因为式(6.3)中存在不等式约束，则原问题和对偶问题都满足KKT条件： \[ \begin{cases}\nabla_{\pmb{w}}=0,\nabla_{b}=0\\ \alpha_i(1-y_i(\pmb{w}^\text{T}\pmb{x}_i+b))=0\\ \alpha_i\geq0\\1-y_i(\pmb{w}^\text{T}\pmb{x}_i+b)\leq0\end{cases} \] 其中的第二个条件被称为slackness条件。可以这样理解，如果\(1-y_i(\pmb{w}^\text{T}\pmb{x}_i+b)<0\)，为了使得\(L(\pmb{w},b,\pmb{\alpha})\)最小，\(\alpha_i\)必然是0，该样本不会出现在(6.12)的求和中，不会对超平面的结果产生任何影响。如果\(1-y_i(\pmb{w}^\text{T}\pmb{x}_i+b)=0\)，若\(\alpha_i>0\)，则\(y_i(\pmb{w}^\text{T}\pmb{x}_i+b)=1\)，所对应的样本点位于最大间隔边界上，是一个支持向量。若\(\alpha_i=0\)，样本点不会对超平面产生影响。这也揭示了支持向量机的一个重要性质：训练完成后，大部分的训练样本都不需要保留，最终模型只与支持向量有关。

    由原始问题的约束条件\(y_i(\pmb{w}^\text{T}\pmb{x}_i+b)=1\)，和支持向量\((\pmb{x}_k,y_k)\)，解得： \[ \exists (\pmb{x}_k,y_k),s.t.\ y_k(\pmb{w}^\text{T}\pmb{x}_k+b)=0\\ b^*=y_k-\stackrel{m}{\underset{i=1}\sum}\alpha_iy_i\pmb{x}_i^\text{T}\pmb{x}_k \]

    至此，我们终于解决了\(\pmb{w},b\)的求解问题。
