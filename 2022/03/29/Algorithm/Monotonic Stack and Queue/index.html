    单调栈

    定义

    单调栈（monotonic stack），顾名思义，即满足单调性的栈结构

    伪代码描述

    insert x
while !sta.empty() && sta.top()<x
    sta.pop()
sta.push(x)

    解决的问题

    定义函数 \(f(i)\) 代表数列中第 \(i\) 个元素之后第一个大于 \(a_{i}\)的元素的下标，即 \(f(i)=min_{i<j\leq n, a_{j} > a_{i}}\{j\}\)。若不存在，则 \(f(i)=0\)。 给定长度为\(n\)的序列，分别求出\(f(1),f(2)\dots f(n)\)的值。

    样例输入

    5
1 4 2 3 5

    样例输出

    2 5 4 5 0

    示例代码

    [USACO06NOV] POJ3250 Bad Hair Day

    #include <stdio.h>
#include <string.h>

#define MAXN 3000000 + 100

struct num {
    int p, val;
}stack[MAXN];

void Print_array(int *a, int n) {
    for (int i = 1; i <= n; ++i) {
        if (i == 1)printf("%d", a[i]);
        if (i > 1)printf(" %d", a[i]);
    }
    printf("\n");
}

int ans[MAXN];
int solve() {
    int top = 0;
    int n, x;
    scanf("%d", &n);
    for (int i = 1; i <= n; ++i) {
        scanf("%d", &x);
        struct num xx = {i, x};
        if (top == 0) stack[++top] = xx, ans[stack[top].p] = 0;
        else {
            while (top && xx.val > stack[top].val) {
                ans[stack[top].p] = xx.p;
                --top;
            }
            stack[++top] = xx;
        }
    }
    while (top--) {
        ans[stack[top].p] = 0;
    }
    Print_array(ans, n);
    return 1;
}
int main() {
    solve();
    return 0;
}

    单调队列

    定义

    单调队列（monotonic queue），即内部元素具有单调性的队列。

    例题：最大子序和

    给定一个长度为\(N\)的整数序列\(a_1,a_2,\dots,a_N\)，从中找出一段长度不超过\(M\)的连续子序列，使得子序列中所有数的和最大。其中\(0\leq M\leq N\leq 5\times10^5,-500\leq a_i\leq 500\)，且保证答案的绝对值在\([0,2^{31}-1]\)之内。

    提交

    • 样例输入
    6 3
1 -2 3 -4 5 -6
    • 样例输出
    5

    解答1

    计算区间和的问题，一般转化为“两个前缀和相减的形式求解。令\(S[i]\)表示序列前\(i\)项的和，我们要求解的问题可以用数学方式描述为： \[ J=\max\{S[i]-\underset{i-M\leq j\leq i-1}{\min}\ S[j]\} \] 对于这种问题，我们可以使用最直观的方法求解，即枚举每个\(i\)，固定\(i\)的值后枚举\(j < i\)，最后获得\(S[i]-S[j]\)的最大值，时间复杂度是\(O(NM)\)的。

    但我们发现，在枚举过程中，在已知当前最大值的情况下，有些\(i,j\)的出现是无意义的。考虑如下的情况，现有\(j<k<i\)，而\(S[j]>S[k]\)，那么，\(j\)这个位置在\(i\)以后的枚举中就可以舍弃掉，因为如果今后枚举到\(S[c](i<c\leq N)\)时，\(S[c]-S[k]\)一定大于\(S[c]-S[j]\)。且\(k\)离\(c\)更近，\(c-k+1\)更不容易超过\(m\)。

    总结上面的分析，可能成为最优选择的策略集合2一定是一个”下标位置递增，对应的前缀和S的值也递增”的序列。我们可以用一个双端队列保存这个序列。随着右端点从前向后扫描。我们对\(i\)进行以下步骤：

    1. 判断队头决策与\(i\)的距离是否超过\(M\)，若超出则出队。
    2. 此时队头就是右端点为\(i\)时，左端点\(j\)的最优选择。
    3. 不断删除队尾决策，直到队尾下标对应的\(S\)值小于\(S[i]\)。然后把\(i\)作为新的决策入队。

    总结

    1. 本题的难点是对决策单调性的发掘。

    2. 使用单调队列优化的前提是找出决策序列的单调性。

    示例代码

    int solve(int n, int m) {
    s[0] = 0;
    for (int i = 1; i <= n; ++i) s[i] = s[i - 1] + a[i];
    int l = 1, r = 1, ans = -INF;
    q[1] = 0;
    for (int i = 1; i <= n; ++i) {
        while (l <= r && i - q[l] > m) l++;
        ans = max(ans, s[i] - s[q[l]]);
        while (l <= r && s[i] <= s[q[r]]) r--;
        q[++r] = i;
    }
    return ans;
}

    总结

    单调栈和单调队列的本质，是借助单调性，及时排除不可能的决策，保持候选集合的高度有效性和秩序性。3
